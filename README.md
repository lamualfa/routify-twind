# routify-twind

Use Twind (_Tailwind CSS in JS version_) in Routify. What is Twind? see [Twind](https://github.com/tw-in-js/twind) for more informations.

## Installation

```bash
npx degit lamualfa/routify-twind project-name
cd project-name
yarn && yarn dev
```

<hr>

## Environment

- Routify - `v2.5.1-next-major`
- Twind - `v0.9.1`
